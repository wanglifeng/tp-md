#### 机器人配置方法

示例：

```js
<a href="https://img.life.cntaiping.com/tpbb/scs/pre/viewPDF/index.html#/?file={太平服务器上文件地址}&name={下载文件名称}">显示给用户看的文件名字</a>
```



- 匹配href地址路径为项目地址，显示pdf 页面样式。
- PDF需要在太平服务下
